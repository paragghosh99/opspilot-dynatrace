# Demo Script

1. Open the OpsPilot dashboard and show the live Dynatrace problem feed.
2. Click `Run poll` to trigger autonomous detection and response.
3. Open the incident history tab and show the generated diagnosis, root cause, and remediation action.
4. Open MTTR trends and show the drop in resolution time across historical incidents.
5. Open Learned Runbook and show how prior incidents update recommended actions.
6. Close by showing the Dynatrace MCP, Gemini, Agent Builder, and Google Cloud services in the architecture.
